# life-game
Life game with web assembly

```
$ source <path-to-emsdk>/emsdk_env.sh
```

## Compile
```
# C++
$ emcc hoge.cpp -std=c++14 -s WASM=1 -o hoge.html

# C
$ emcc hoge.c -s WASM=1 -o hoge.html 
```

## Run
```
$ emrun hoge.html
```

## Required
- \>= C++14
- Emscripten
